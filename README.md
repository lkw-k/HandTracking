# Hand Tracking - 손 제스처로 마우스 제어

웹캠으로 손을 인식하고, 손가락 제스처로 마우스 동작을 자동 제어하는 프로젝트입니다.

## 기능

### 손가락 제스처 인식
| 제스처 | 기능 |
|--------|------|
| **검지만 펴짐** | 마우스 이동 |
| **주먹 쥐기** | 마우스 드래그 (지속) |
| **검지+중지 끝 접촉** | 좌클릭 |
| **엄지+검지+중지 펴짐** | 우클릭 |
| **검지+중지+약지+소지 펴짐** | 스크롤 업 |
| **중지+약지+소지 펴짐** | 스크롤 다운 |
| **5손가락 전부 펴짐** | 마우스 고정 |

## 요구 사항

- Python 3.11 이상
- 웹캠
- Windows/Mac/Linux

## 설치

```bash
# Python 버전 확인
python --version

# 의존성 설치
pip install -r requirements.txt
# 또는 uv 사용
uv sync
```

### 주요 라이브러리
- **mediapipe (0.10.9)**: 손 인식 및 랜드마크 감지
- **opencv-python**: 웹캠 영상 처리 및 시각화
- **pyautogui**: 마우스/키보드 제어

## 사용 방법

```bash
python main.py
```

1. 웹캠 창이 열립니다
2. 손을 카메라에 보여줍니다
3. 손가락 제스처로 마우스를 제어합니다
4. `p` 키를 눌러 종료합니다

## 주요 파라미터

- **해상도**: 320x240 (낮은 해상도로 성능 최적화)
- **프레임 레이트**: 120 FPS
- **마우스 스무딩**: 0.23 (마우스 움직임을 부드럽게 함)

## 알려진 제한사항

- 한 손만 인식 (최대 1개 손)
- 충분한 조명이 필요합니다
- 손이 카메라 범위를 벗어나면 드래그 자동 해제
